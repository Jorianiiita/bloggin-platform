# bloging-platform
A platform to start blogging articles.

## Setup
1. Clone the repository `git@github.com:Jorianiiita/bloging-platform.git`.
2. Install npm and node latest version.
3. Run npm install.
```sh
$ npm install
```
4. Run npm run start.
```sh
$ npm run start
```
5. Goto `http://localhost:8001` to see bloging-platform.
